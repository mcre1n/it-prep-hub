## 📋 О проекте

**IT Prep Hub** — это платформа для подготовки к IT-собеседованиям с тестами и практическими заданиями.

### 🎯 Цели проекта для портфолио

Этот проект демонстрирует мои навыки как **DevOps инженера**:

| Область | Что реализовано |
|---------|-----------------|
| **CI/CD** | GitHub Actions (авто-тесты, авто-деплой) |
| **Контейнеризация** | Docker, docker-compose |
| **Оркестрация** | Kubernetes (minikube для разработки) |
| **IaC** | Terraform (Google Cloud Platform) |
| **Мониторинг** | Sentry, Firebase Performance, Google Analytics |
| **Безопасность** | CodeQL, Dependency scanning |
| **Infrastructure** | Firebase Hosting, Firestore, Authentication |
